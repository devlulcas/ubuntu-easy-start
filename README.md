# :scroll: UBUNTU EASY(IER) START

Give the entry files executable permissions:

```bash
chmod +x init.sh
chmod +x fonts.sh
chmod +x apps.sh
```

Run `init.sh` first:

```bash
sudo ./init.sh
```

Then run `fonts.sh`:

```bash
sudo ./fonts.sh
```

Finally, run `apps.sh`:

```bash
sudo ./apps.sh
```

## :warning: WARNING

This script is intended for Ubuntu 22.04 LTS. It may not work on other versions of Ubuntu.
